# Clínica Veterinaria - Base de Datos

Este proyecto consiste en la creación de una base de datos para gestionar una clínica veterinaria.

## Contenido
- Creación de tablas
- Inserción de datos
- Consultas SQL (JOIN, COUNT)
- Actualización y eliminación de datos
- Uso de transacciones (BEGIN, COMMIT)

## Tecnologías
- PostgreSQL
- pgAdmin
